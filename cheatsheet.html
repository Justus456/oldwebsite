Website Quick Guide

A handy quick reference for some common HTML and CSS elements.

HTML
Tag Notes
<!DOCTYPE html> First line of all HTML documents to set standard in avoid “quirks mode”
<html> Everything must be inside these tags
<head> Website metadata; link to styesheets in these tags
<link href=“styles.css” type=“text/css” rel=“stylesheet”> Stylesheet link
<meta name="viewport" content="width=device-width, initial-scale=1"> Set the viewport size
<body> All other content is contained within these tags
<header> Header structural element
<footer> Footer structural element
<h1> to <h6> Header text; 6 available header tags
<p> Paragraph text
<img src=""> Image
<table> Table element; all other table elements contained within these tags
<thead> Table header structural element
<tr> Table row
<th> Table header cell
<tbody> Table body structural element
<td> Table data, or cell

CSS
Selectors
Syntax Notes
type Select HTML tag type
.class Select class; class name is preceded by period
#id Select id; id name is preceded by hash sign
* Universal selector
@media only screen and (min-width: 320px) and (max-width: 480px) Rules for specific screen sizes (e.g. mobile screen, tablet, etc.); adjust sizes as necessary
Properties
Property Notes
background-color Gives background color to selected element; hex, rgb, hsv, common name
background-image Gives selected element background image; needs src
border Assigns border to selected element; size weight color
color Color of selected element; hex, rgb, hsv, common name
display -
width -
height -
font-family Assign font family to selected element
font-style Change font style of selected element
font-size Change font size of selected element
font-weight Change font weight of selected element
margin Adjust margin of selected element
padding Adjust padding of selected element
opacity Change opacity of selected element
position -

Javascript
Operators
Operator Notes
+ Addition
++ Increment by 1
= Assignment
=== Strict Equality (Loosely Equal == will attempt to convert different data types)
/ Division
> < Greater Than and Less Than; inlude = for Greater Than/Less Than or Equal To
* Multiplication
% Remainder
- Subtraction
&& Logical And
|| Logical Or
! Logical Not (e.g. != and !== for Inequality and Strict Inequality)
Objects
Object Code Example
Array = [] const arrayName = [arrayItem0, arrayItem1, ...]
arrayName[0]
Function() {} Or () => {} function(var1, var2) {return var1 + var2;}
(var1, var2) => return var1 + var2;
Statements
Statement Notes
const Declare variable; cannot be reassigned, but object properties can be modified
let Declare variable that can be reassigned
if () {} else {} If (conditions in parenthesis are true) {code in block will execute} else {code in block will execute}
return Ends function execution and specifies a value to be returned to the function
